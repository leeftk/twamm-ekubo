# Starknet Deployment Scripts

## Account Setup and Deployment
# Create a new account
sncast account create \
    --url https://free-rpc.nethermind.io/sepolia-juno \
    --name some-name

# Deploy the account
sncast account deploy \
    --url https://free-rpc.nethermind.io/sepolia-juno \
    --name some-namee \
    --fee-token strk
